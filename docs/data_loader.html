data_loader.intent_prediction_dataset.compute_normalization_stats(dataset)[source]

Computes normalization statistics (mean and standard deviation) for the given dataset.

The function extracts multiple features from the dataset’s sequences including robot positions, velocities, speed, bounding boxes, positional data, distance, and actions. These features are concatenated to form a feature matrix, from which the mean and standard deviation are computed along each feature dimension to facilitate normalization.

Parameters:

dataset – A dataset object containing sequences of data samples. Each sample is expected to have fields such as ‘robot_position’, ‘robot_velocity’, ‘bounding_box’, ‘position’, ‘distance’, and ‘action’.

Returns:

A tuple containing two numpy arrays:

  • mean: The mean of each feature across all sequences in the dataset.

  • std: The standard deviation of each feature across all sequences in the dataset.

Return type:

Tuple[np.ndarray, np.ndarray]

data_loader.multi_modal_dataset.compute_normalization_stats(dataset)[source]

Computes the normalization statistics (mean and standard deviation) for the features extracted from the provided dataset. The function processes all samples in the dataset, retrieves the features for each sample, and calculates the mean and standard deviation across all features.

Parameters:

dataset – The dataset object that provides access to samples and a feature extraction mechanism. It is expected to contain a samples attribute providing sequence data and an _extract_features method to extract features for each sequence.

Returns:

A tuple where the first element is a NumPy array

representing the mean values of the features, and the second element is a NumPy array representing the standard deviation of the features.

Tuple[np.ndarray, np.ndarray]

Splits the dataset into training, validation, and test sets.

This function loads the dataset, computes normalization statistics, shuffles the data, and splits it into training, validation, and test subsets based on the specified ratios.

  • sequence_length (int, optional) – Length of the sequences to extract. Defaults to 16.

  • normalize (bool, optional) – Whether to normalize the dataset. Defaults to True.

  • val_ratio (float, optional) – Proportion of the dataset to use for validation. Defaults to 0.2.

  • test_ratio (float, optional) – Proportion of the dataset to use for testing. Defaults to 0.1.

  • seed (int, optional) – Random seed for reproducibility. Defaults to 42.
